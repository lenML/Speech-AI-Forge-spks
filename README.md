# Speech-AI-Forge-spks

这里是 [Speech-AI-Forge](https://github.com/lenML/Speech-AI-Forge) 这个项目的音色库，这里存放一些公开可用的音色

## 目录简述
- spks: 文件夹放 spk 文件
- docs: 文件夹放文档
- scripts: 一些本库用的脚本
- index.json: 入口，其中配置了所有公开音色，其中有 spk 基本信息和对应下载地址，由脚本生成，一般不手动修改
